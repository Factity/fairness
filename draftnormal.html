1. Exploding and Vanishing Gradients

In deep networks, gradients are computed via backpropagation using the chain rule. This leads to repeated multiplication of Jacobian terms across layers.

For a feedforward network, the gradient w.r.t. weights in layer \( l \) is:

\[ \frac{\partial L}{\partial W^{(l)}} = \frac{\partial L}{\partial a^{(l)}} \cdot \sigma'(z^{(l)}) \cdot a^{(l-1)\top} \]
Consequences of Vanishing Gradients:
  • Early layers stop learning
  • Training plateaus
  • Poor feature representation
Consequences of Exploding Gradients:
  • Instability and divergence
  • NaN values during training
  • Catastrophic weight updates

2. Normalization Techniques

Batch Normalization

For a mini-batch \( \mathcal{B} \): \[ \mu_{\mathcal{B}} = \frac{1}{m}\sum_{i=1}^{m} x_i, \quad \sigma_{\mathcal{B}}^2 = \frac{1}{m}\sum_{i=1}^{m} (x_i - \mu_{\mathcal{B}})^2 \] \[ \hat{x}_i = \frac{x_i - \mu_{\mathcal{B}}}{\sqrt{\sigma_{\mathcal{B}}^2 + \epsilon}} \] \[ y_i = \gamma \hat{x}_i + \beta \]

Layer Normalization

For a single sample with features \( x = (x_1, \dots, x_H) \): \[ \mu = \frac{1}{H}\sum_{i=1}^{H} x_i, \quad \sigma^2 = \frac{1}{H}\sum_{i=1}^{H} (x_i - \mu)^2 \] \[ \hat{x}_i = \frac{x_i - \mu}{\sqrt{\sigma^2 + \epsilon}} \]

3. Application to Different Architectures

Architecture Recommended Normalization Reason
Feedforward Networks (MLP) BatchNorm Large batch sizes available
Convolutional Networks (CNN) BatchNorm Works well with spatial dimensions
Recurrent Networks (RNN/LSTM) LayerNorm Independent of batch size and sequence length
Transformers LayerNorm (Pre-norm) Standard in modern architectures

4. Geometry of the Parameter Space

The loss landscape of neural networks is a high-dimensional Riemannian manifold. The choice of metric significantly affects optimization dynamics.

Natural Gradient Descent uses the Fisher Information Matrix \( F \) as the Riemannian metric: \[ \tilde{\nabla} L(\theta) = F(\theta)^{-1} \nabla L(\theta) \]

Normalization techniques (especially Weight Normalization and LayerNorm) improve the conditioning of the optimization problem by making the parameter space more isotropic.

5. Invariance Properties

Key Insight

Layer Normalization introduces scale invariance per example, while Batch Normalization provides statistical stabilization across the mini-batch.
