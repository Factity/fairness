Layer 01 · Input Representation

TOKEN EMBEDDING SPACE

Every token is first looked up in a learned embedding matrix and projected into a high-dimensional vector space. Semantically related tokens drift toward each other — the geometry of meaning before any attention has run.

Layer 02 · Order Injection

POSITIONAL ENCODING

Attention has no inherent sense of order, so sinusoidal signals at geometrically increasing wavelengths are added to every embedding — letting the model recover relative and absolute position from phase alone.

SIN / COS · 2i / d_model POS 0 – 64

Layer 03 · Scaled Dot-Product Attention

SELF-ATTENTION

Queries and Keys are dotted, scaled, masked, and passed through softmax to produce an attention matrix — each row a probability distribution over which previous tokens the current token should listen to. The pattern reshapes per head.

HEAD 1 / 8 CAUSAL MASK ON

Layer 04 · Full Forward Pass

THE DECODER BLOCK PIPELINE

Tokenize, embed, add position, then loop through N identical decoder blocks — each a pre-norm attention sub-layer and a pre-norm feed-forward sub-layer, both wrapped in a residual add — before a final projection to vocabulary logits.

× N LAYERS AUTOREGRESSIVE

Layer 05 · Position-Wise MLP

FEED-FORWARD NETWORK

Every position is independently expanded to roughly four times the model width, pushed through a GELU nonlinearity, then projected back down — the part of the network most often credited with storing factual knowledge.

d_model → 4× → d_model GELU

Layer 06 · The Information Highway

RESIDUAL STREAM

A single additive pathway runs the full depth of the network. Attention and feed-forward sub-layers each read from it and write a delta back — nothing is overwritten, which is why interpretability researchers treat it as the model's shared workspace.

ADDITIVE · UNNORMALIZED 12 LAYERS SHOWN

ACTIVATION HEAT ACROSS DEPTH

Probing hidden states at every layer reveals a rough division of labor — early layers track syntax and local structure, middle layers carry induction and copying behavior, late layers sharpen toward the final vocabulary distribution.

Layer 08 · Cross-Token Routing

ATTENTION ROUTING MAP

Drawn as arcs over the token sequence, attention edges reveal recognizable circuits — induction heads copying from earlier repeats, previous-token heads, and broad heads that attend back to the sequence start.

Layer 09 · Where the Time Goes

INFERENCE TIMING DOMAINS

Prefill is compute-bound and parallel across the prompt; decode is memory-bound and strictly sequential, one token at a time, dominated by reading the growing KV cache back off HBM.

Layer 10 · From Logits to Token

TOKEN SAMPLING

The final linear layer projects hidden state to vocabulary-sized logits, softmax turns them into a probability cloud, and temperature plus top-k / top-p trims the field before one token is drawn — then fed back in to start the next step.
